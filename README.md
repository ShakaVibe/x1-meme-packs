# X1 Meme Packs

**Bridge. Earn. Collect.**

Bridge USDC from Solana to X1 and receive an instant airdrop of the hottest meme coin pack in crypto. The more you bridge, the rarer your pack.

## Pack Tiers
| Bridge Amount | Tier | Multiplier |
|---|---|---|
| $100 – $499 | 🥉 Bronze | 1x |
| $500 – $999 | 🥈 Silver | 3x |
| $1,000 – $1,999 | 🥇 Gold | 7x |
| $2,000 – $4,999 | 💎 Diamond | 15x |
| $5,000 | 🔥 Legendary | 30x |

## Features
- Instant pack airdrop after bridging
- 10 meme tokens in every pack
- Daily pack upgrade rule (always keep your best tier)
- Admin panel for managing reward tokens
- Built on X1 Blockchain via X1 Warp Bridge

## Built With
- HTML / CSS / JavaScript
- X1 Warp Bridge integration
